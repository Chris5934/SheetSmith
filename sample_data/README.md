# Sample Data

This directory contains example data demonstrating SheetSmith's capabilities.

## Contents

- `example_spreadsheet.json` - Simulated spreadsheet export with formulas
- `example_patches/` - Example patch files showing before/after changes
- `logic_blocks/` - Common logic block patterns

## Example Spreadsheet Structure

The example spreadsheet simulates a damage calculator with:
- Multiple character build sheets
- Shared kit logic (damage ratios, status effects)
- Teammate buff calculations
- Rotation timing

## Usage

These files can be used to:
1. Test the formula analyzer without connecting to Google Sheets
2. Understand the expected data formats
3. Learn how patches are structured
